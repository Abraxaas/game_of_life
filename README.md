# Игра жизни 1.0

Тестовая онлайн-версия 1.0 для геймификации рутины и полезных привычек в формате спокойной ролевой прокачки.

Пользователь создает квесты, привязывает их к жизненным статам, отмечает выполнение и получает прогресс без облака, регистрации и сервера.

## Стек

- React
- TypeScript
- Vite
- чистый CSS
- IndexedDB для хранения данных в браузере
- `localStorage` только для мелкой UI-настройки активной вкладки

## Что умеет версия 1.0

- показывает профиль и общий прогресс
- хранит 5 базовых статов из конфигурации
- поддерживает создание, редактирование, архивирование и удаление квестов
- поддерживает `daily` и `one_time` квесты
- начисляет опыт по сложности
- пересчитывает уровни статов и общий прогресс профиля
- считает текущую и лучшую серию для daily-квестов
- показывает историю выполнений
- умеет экспортировать и импортировать все данные в JSON
- имеет seed-данные для первого запуска
- работает без backend и внешних API

## Структура проекта

```text
src/
  app/                корневой shell и React context
  pages/              страницы главного экрана, истории и резервной копии
  components/         общие UI-компоненты
  features/
    quests/           список квестов и модалка формы
    stats/            профиль и карточки статов
    history/          экран истории
    backup/           экспорт, импорт и настройки
  shared/             константы и конфиг статов
  storage/            IndexedDB, normalizer, seed-данные
  utils/              опыт, уровни, серии, даты, валидация импорта
  types/              доменные типы
```

## Где хранятся данные

- основное хранилище: `IndexedDB`
- имя базы: `routine-rpg-mvp`
- вспомогательная UI-настройка активной вкладки: `localStorage`

Если `IndexedDB` недоступна, приложение показывает предупреждение и временно работает в памяти без постоянного сохранения.

## Экспорт и импорт

Экспорт:

- открой вкладку `Резервная копия и настройки`
- нажми `Экспортировать данные`
- приложение скачает JSON-файл со всем состоянием пользователя

Импорт:

- открой вкладку `Резервная копия и настройки`
- выбери JSON-файл через `Импортировать JSON`
- приложение проверит структуру данных и заменит текущее состояние в браузере

Защита:

- битый JSON не роняет приложение
- структура файла валидируется перед импортом
- импорт более новой версии схемы отклоняется с понятным сообщением

Базовые статы задаются в файле:

## Формула уровней

- уровень `1 -> 2`: `100 опыта`
- далее порог увеличивается на `25 опыта` за каждый следующий уровень

Логика лежит в:

- [src/utils/xp.ts](/Users/dmitrijmalysev/Documents/prog/game/codex_repo/src/utils/xp.ts)

## Ограничения версии 1.0

- нет облачной синхронизации
- нет аккаунтов и авторизации
- нет сервера и уведомлений
- нет мобильной адаптации mobile-first, только базовая адаптация под узкое окно
- нет восстановления удаленных квестов
- нет сложной миграции старых версий импорта, кроме базовой проверки версии схемы
